Ratio Calculator (비율계산기)

Usage Guide (이용 가이드)

  1. Automatically calculates the ratio in the form A : B = C : D.
  2. If you enter exactly 3 out of the 4 fields, the remaining field will be calculated automatically.
  3. If all 4 fields are filled, a warning message will be displayed.
  4. Only numeric input is allowed.
  5. Use cases: resizing photos, calculating video aspect ratios, etc.
